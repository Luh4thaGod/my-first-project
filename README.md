# 🍔 Lezzet Durağı

This project is a modern restaurant menu and ordering interface built using HTML, CSS, and JavaScript.
It was created with AI assistance and later customized and improved by me.

## 🚀 Features

* Category-based product filtering
* Dynamic product listing
* Add to cart functionality
* Total price calculation
* Simple checkout interface
* Modern and responsive design

## 🎯 Purpose

The goal of this project is to improve my frontend development skills and understand real-world project logic.

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📌 Note

This project was created for learning purposes and is open for further improvements.
